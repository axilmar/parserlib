Detailed Description

template<class ParseNode>
class parserlib::logical_not_parse_node< ParseNode >

A parse node which invokes a child node, then restores the parse context state, returning the opposite of the result of the child node. It can be used as a logical predicate.

Parameters
ParseNodetype of child parse node.
