Detailed Description

template<class ParseContext = parse_context<>>
class parserlib::rule< ParseContext >

A rule represents a piece of grammar that can be recursive. Rules in grammar trees are manipulated via references.

Parameters
ParseContextthe parse context to use for this rule.
