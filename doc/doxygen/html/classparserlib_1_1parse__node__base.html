◆ init()

void parserlib::parse_node_base::init ( ) const
inline

Does dynamic initialization of the parse node. It debug mode, it sets the m_text member of the parse node, if not set yet. Setting the text dynamically is required due to rules being recursive. The operation is thread-safe (protected by a recursive mutex, avoiding recursive rules infinite recursion). In release mode, it does nothing. This function will be invoked automatically from parse_context::parse(parse_node).
