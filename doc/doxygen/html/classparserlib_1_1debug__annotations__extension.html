Detailed Description

template<class OutputStream = std::ostream>
class parserlib::debug_annotations_extension< OutputStream >

A parse context extension that allows printing of annotations to an output stream, while parsing.

Parameters
OutputStreamtype of output stream to put the information on; by default, it is std::cout, for std::ostream, and std::wcout for std::wostream.
